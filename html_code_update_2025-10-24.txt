To automatically refresh or update an HTML page, you can use either the <meta> tag or JavaScript. Here are examples of both methods:

1. Using the <meta> Tag
This method is simple and doesn't require JavaScript. Add the following line inside the <head> section of your HTML file:
Html<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="refresh" content="10">
  <title>Auto Refresh</title>
</head>
<body>
  <h1>This page refreshes every 10 seconds!</h1>
</body>
</html>


The content="10" attribute specifies the refresh interval in seconds.


2. Using JavaScript
This method provides more flexibility and control over the refresh behavior:
Html<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Auto Refresh with JavaScript</title>
</head>
<body>
  <h1>This page refreshes every 10 seconds using JavaScript!</h1>
  <script>
    setTimeout(() => {
      location.reload();
    }, 10000); // Refresh every 10,000 milliseconds (10 seconds)
  </script>
</body>
</html>


The setTimeout function reloads the page after the specified time (in milliseconds).


3. Using JavaScript with setInterval
If you want to perform specific updates without reloading the entire page, you can use setInterval:
Html<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Dynamic Content Update</title>
</head>
<body>
  <h1>Dynamic Content Update Example</h1>
  <p id="time"></p>
  <script>
    setInterval(() => {
      const currentTime = new Date().toLocaleTimeString();
      document.getElementById('time').textContent = `Current Time: ${currentTime}`;
    }, 1000); // Update every 1 second
  </script>
</body>
</html>


This updates only the content of the page (e.g., the current time) without refreshing the entire page.
